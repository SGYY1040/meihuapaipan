# 🌸 梅花易数排盘工具 (MeiHua Lite)

> [👉 点击这里，立即开始排盘](https://sgyy1040.github.io/meihuapaipan/)

一款轻量级、零依赖的纯静态网页排盘工具，专注于提供“极速、纯粹、私密”的易学体验。

### 核心特性
*   **🔢 四重起卦模式**：支持【时间起卦】、【报数起卦】（1-3数）、【随机起卦】及【手动装卦】，满足不同场景需求。
*   **🎴 卡片式 UI 设计**：现代化审美，排盘结果以清晰的卡片呈现，重点突出体用、变爻与互卦。
*   **💾 本地化数据存储**：拒绝联网上传，所有历史记录存储于浏览器本地 (LocalStorage)，绝对私密安全。
*   **📱 极速响应**：单 HTML 文件架构，秒开无卡顿，完美适配手机端与 PC 端。
*   **📂 备份与恢复**：支持将历史记录一键导出为 JSON 备份，数据永久掌握在自己手中。
